# Security Policy

## API Keys

Do not paste real CCGO image MCP keys into issues, pull requests, screenshots, chat logs, or public files.

The installer stores the key locally in Codex / Claude MCP configuration files so the local client can call the MCP server later. Rotate the key in CCGO if you believe it has been exposed.

## Reporting Issues

For security issues, contact CCGO support privately instead of opening a public GitHub issue with sensitive details.

When reporting a bug publicly, remove:

- API keys
- local user paths that reveal private information
- generated content you do not want public
- screenshots that include keys or account data
