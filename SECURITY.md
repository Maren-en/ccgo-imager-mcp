# Security Policy

## 中文说明

不要把真实的 CCGO 生图 MCP Key 发到 GitHub issue、pull request、截图、聊天记录或公开文件里。

安装器会把 Key 保存在你本机的 Codex / Claude MCP 配置文件中，这样本机客户端后续才能自动调用 MCP。请不要公开这些配置文件或包含 Key 的截图。

如果你怀疑 Key 已经泄露，请先在 CCGO 后台停用旧 Key，再重新安装并写入新 Key。

如果要反馈安全问题，请私下联系 CCGO 支持，不要在公开 issue 中粘贴敏感信息。

## English

Do not paste real CCGO image MCP keys into issues, pull requests, screenshots, chat logs, or public files.

The installer stores the key locally in Codex / Claude MCP configuration files so the local client can call the MCP server later. Do not publish these configuration files or screenshots that include keys.

If you believe a key has been exposed, rotate it in CCGO and reinstall this MCP with the new key.

For security issues, contact CCGO support privately instead of opening a public GitHub issue with sensitive details.
